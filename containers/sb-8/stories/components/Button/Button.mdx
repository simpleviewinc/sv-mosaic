## Popover

When using `popover` if you need to programmatically close it, you can do so by using `onClose` provided by the React context API.

```js
import * as React from "react";
import { useContext } from "react";
import { Button, ButtonPopoverContext } from "#mosaic";

function MyComponent(props) {
	const buttonPopoverContext = useContext(ButtonPopoverContext);
	// ...rest of component
}
```
