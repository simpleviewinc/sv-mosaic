# Menu

`Menu` is a component that allows you to invoke a dropdown of MenuItems. Each menuItem can have it's own unique `onClick` handler.

## Props

* anchorEl - HTMLElement
* open - boolean
* onClose - function
* items - Array of [MenuItemProps](https://github.com/simpleviewinc/sv-mosaic/blob/develop/src/components/MenuItem/MenuItemTypes.ts)
