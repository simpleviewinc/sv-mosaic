# MenuSelect

`MenuSelect` is responsible for rendering the content of `Menu` with a behavior similar to a native `<select>`.

* If the trigger for the menu is not a button, you can invoke it via placing it inside `MenuBase`.
* If the trigger is a `Button`, then pass it to `menuContent`.
